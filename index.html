Selectors

A selector is a pattern that matches one or more elements in a document. Selectors are used to apply styles to elements based on their name, id, class, attributes, and position in the HTML document tree.

Here are some commonly used selectors:

  • Type Selector
  • Class Selector
  • ID Selector
  • Attribute Selector
  • Descendant Selector

Here's an example:


          p {
            color: red;
          }
Properties

A property is a set of values assigned to an element. Properties are used to control the presentation and behavior of an element.

Here are some commonly used properties:

  • Background-color
  • Color
  • Font-family
  • Width
  • Height

Here's an example:


          p {
            color: red;
            font-size: 16px;
          }
Values

A value is a single piece of data that can be assigned to a property. Values are used to define the appearance and behavior of an element.

Here are some commonly used values:

  • Color values
  • Length values
  • Percentage values
  • Keyword values
  • Unitless values

Here's an example:


          p {
            color: red;
            font-size: 16px;
            width: 50%;
          }
Box Model

The box model refers to the way elements are displayed on a web page as rectangular boxes, and how each box's content, padding, border, and margin are defined.

The box model consists of four parts:

  • Content: The actual content of the box, which can be text, an image, or other HTML elements.
  • Padding: The space between the content and the border.
  • Border: The border surrounding the content and padding.
  • Margin: The space outside the border, between the border and other elements on the page.

Here's an example of how the box model works:


            .example {
              width: 200px;
              height: 100px;
              padding: 20px;
              border: 2px solid black;
              margin: 10px;
            }

In the example above, the box has a width of 200 pixels and a height of 100 pixels. It also has 20 pixels of padding on all sides, a 2-pixel border that is solid and black, and a margin of 10 pixels on all sides. The total size of the box, including its content, padding, border, and margin, is 244 pixels wide and 144 pixels tall.

Layouts

In web design, layouts are used to arrange elements on a web page. Layouts can be created using CSS, and are used to define the structure and flow of a page's content.

Here are some commonly used layout techniques:

  • Floats
  • Flexbox
  • Grid
  • Positioning

Each of these techniques has its own strengths and weaknesses, and can be used to create a variety of different layouts.

Here's an example of a simple layout using flexbox:


            .layout-example {
              display: flex;
              justify-content: space-between;
            }
          .box {
          height: 100px;
          width: 100px;
          }
          
          .red {
          background-color: red;
          }
          
          .green {
          background-color: green;
          }
          
          .blue {
          background-color: blue;
          }
